# Prompt para Agente de Datos Ecuador

Identifica fuentes trimestrales para Ecuador 2000-I a 2024-IV: PIB real, exportaciones por sector/producto, FBKF, importaciones, precio del petróleo, apertura comercial y empleo. Prioriza Banco Central del Ecuador. Documenta fuente, enlace, frecuencia, unidad, cobertura y limitaciones. No uses datos anuales si no existe justificación.

Entrega: `02_datos/metadata/fuentes_datos.md` y `diccionario_datos.md`.
