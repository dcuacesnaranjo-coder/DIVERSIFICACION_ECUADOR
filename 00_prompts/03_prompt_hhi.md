# Prompt para Agente Constructor del IHH

Con la base sectorial trimestral, calcula el índice Herfindahl-Hirschman: HHI = sum(s_i^2), donde s_i es la participación del sector i en el total. Genera una base con trimestre, HHI, número de sectores y clasificación. Grafica la evolución 2000-I a 2024-IV. Explica que mayor HHI significa mayor concentración.
