# Prompt para Agente Investigador de Literatura

Construye una matriz de literatura para el tema de diversificación productiva, crecimiento económico y desarrollo sostenible. Incluye papers que usen HHI, ARDL, VECM, cointegración, quiebres estructurales y economías dependientes de recursos naturales. Clasifica cada paper por país, método, variable dependiente, medida de diversificación, hallazgos y utilidad para Ecuador.

Entrega: `01_literatura/matriz_literatura.md` y actualización de `referencias_apa.md`.
