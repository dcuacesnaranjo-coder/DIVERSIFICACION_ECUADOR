# Prompt para Agente Econometrista

Usa la base trimestral final para evaluar la relación entre HHI y crecimiento económico. Aplica ADF, PP y KPSS. Verifica que no existan variables I(2). Si las variables son mezcla I(0)/I(1), estima ARDL Bounds Test. Si las variables son I(1) y hay cointegración clara, estima VECM como alternativa. Incluye diagnósticos de autocorrelación, estabilidad, heterocedasticidad y normalidad.

Entrega resultados en `04_resultados/modelos/` y una interpretación económica clara.
