# Diversificación productiva, crecimiento económico y desarrollo sostenible en Ecuador bajo dolarización

## Tema del proyecto
**Diversificación productiva, crecimiento económico y desarrollo sostenible en Ecuador bajo dolarización: un análisis econométrico trimestral mediante el índice Herfindahl-Hirschman, período 2000-2024.**

## Línea y sublíneas
- **Línea de facultad:** Administración y Economía para el desarrollo sostenible de las organizaciones.
- **Sublínea principal:** Economía y Desarrollo sostenible.
- **Sublínea secundaria:** Economía, sociedad y políticas públicas.

## Paper base
Siddiqui, S. A., & Afzal, M. N. I. (2022). *Sectoral diversification of UAE toward a knowledge-based economy*. Review of Economics and Political Science, 7(3), 177–193.

Este paper sirve como base conceptual y metodológica parcial porque utiliza el índice Herfindahl-Hirschman para medir diversificación sectorial en una economía dependiente de recursos naturales. La réplica para Ecuador no debe copiar el DEA como método principal; debe adaptar el HHI y estimar la relación dinámica con crecimiento económico mediante series de tiempo trimestrales.

## Objetivo general
Analizar la incidencia de la diversificación productiva, medida mediante el índice Herfindahl-Hirschman, sobre el crecimiento económico del Ecuador bajo dolarización durante el período 2000-I a 2024-IV, mediante evidencia econométrica trimestral, con el fin de aportar criterios para políticas de desarrollo sostenible.

## Estrategia metodológica resumida
1. Construir una base trimestral 2000-I a 2024-IV.
2. Calcular el IHH trimestral por estructura exportadora o productiva.
3. Describir la evolución de la concentración/diversificación.
4. Estimar relación con crecimiento económico usando ARDL si hay mezcla I(0)/I(1).
5. Usar VECM solo si todas las variables relevantes son I(1) y existe cointegración clara.
6. Controlar quiebres estructurales: 2008-2009, 2014-2016, 2020 y otros shocks documentados.
7. Redactar paper con resultados reproducibles.

## Regla de oro
No trimestralizar datos anuales de forma artificial para inflar muestra. Si una variable no existe trimestralmente, se reemplaza por una proxy válida o se justifica su exclusión.
