# Fuentes de datos candidatas

## Banco Central del Ecuador
- Cuentas Nacionales Trimestrales.
- Producto Interno Bruto real.
- Formación Bruta de Capital Fijo.
- Exportaciones e importaciones por producto/sector.
- Precio del petróleo ecuatoriano si está disponible.

## INEC
- ENEMDU trimestral para empleo, si la serie es comparable.

## Fuentes internacionales
- World Development Indicators: respaldo anual, no fuente principal si se exige trimestral.
- CEPALSTAT: respaldo regional.
- FRED/EIA: precio internacional del petróleo.
- UN Comtrade/OEC/Atlas: útil para comercio, pero verificar frecuencia.

## Regla
La fuente principal debe permitir replicar datos trimestrales 2000-I a 2024-IV.
