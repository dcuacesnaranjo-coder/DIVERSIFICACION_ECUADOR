# Checklist de reproducibilidad

- [ ] Existe fuente documentada para cada variable.
- [ ] Los datos raw no fueron editados manualmente.
- [ ] El IHH se calcula desde participaciones sectoriales reales.
- [ ] El crecimiento económico se mide como variación o log-diff del PIB real.
- [ ] Se ejecutaron pruebas ADF/PP/KPSS.
- [ ] Se verificó ausencia de variables I(2) antes de ARDL.
- [ ] Se justificó VECM solo si hubo cointegración.
- [ ] Se controlaron quiebres estructurales.
- [ ] Las tablas del paper provienen de archivos reproducibles.
- [ ] Las figuras del paper provienen de scripts.
- [ ] Las conclusiones no exceden la evidencia.
