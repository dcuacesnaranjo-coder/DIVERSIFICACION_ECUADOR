# 3. Metodología

## 3.1 Datos
Datos trimestrales 2000-I a 2024-IV. La fuente principal será el Banco Central del Ecuador para PIB real, exportaciones, importaciones, FBKF y otros agregados.

## 3.2 Índice Herfindahl-Hirschman
El HHI se calcula como la suma de las participaciones sectoriales al cuadrado. Un valor más alto implica mayor concentración y menor diversificación.

## 3.3 Estrategia econométrica
Se aplicarán pruebas ADF, PP y KPSS. Si las variables son I(0)/I(1), sin I(2), se estimará ARDL. Si existe cointegración clara entre variables I(1), se considerará VECM. Se incorporarán pruebas de quiebre estructural.
